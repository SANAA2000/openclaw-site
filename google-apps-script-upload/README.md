# Google Apps Script upload page

Files ready to paste into a Google Apps Script project:

- `Code.gs`
- `Index.html`

## Deploy steps
1. Open https://script.google.com
2. Create a new project
3. Replace `Code.gs` with the provided file
4. Add an HTML file named `Index` and paste `Index.html`
5. Deploy > New deployment > Web app
6. Execute as: Me
7. Who has access: Anyone with the link (or Anyone)
8. Authorize access to Drive when prompted
