# My Web Project

A clean, modern, responsive landing page built with plain HTML, CSS, and JavaScript.

## Files

- `index.html` - main landing page structure
- `style.css` - responsive styles and layout
- `script.js` - small enhancement for dynamic footer year

## Features

- Responsive layout for desktop, tablet, and mobile
- Modern typography and spacing
- Hero, About, Services, Contact, and Footer sections
- Rounded service cards
- Button hover effect
- Smooth scrolling
- Works directly as a static website

## Deployment

This project is ready for static hosting on platforms like Vercel, GitHub Pages, or Netlify.
